# 第 9 章 最佳解释与因果推理：论证与写作策略

拿破仑是死于下毒吗？莎士比亚是否打算将伊阿古（Iago）塑造成一个通俗剧式的反派？阿琳（Arlene）那一刀的力度是否只是中等？冥王星曾是海王星的卫星吗？海豹岛（Seal Island）地下是否有大量石油沉积？首相打算近期辞职吗？玛蒂娜（Martine）喜欢卡尔（Karl）吗？弗兰克（Frank）是故意绊倒贝利（Pele），还是纯属意外？总统感到不安吗？汽车为什么会撞上电线杆？恐龙灭绝的原因是什么？托马斯·杰斐逊所说的“平等”是什么意思？

## 最佳解释假设

针对这些五花八门的问题，所提出的任何答案都是一个**解释事实的假设**（或结论），其合理性主要取决于它解释事实的能力，以及其竞争对手无法解释这些事实——或者用一个技术术语来说，它是**最佳解释**（best explanation）。遗憾的是，这些问题的答案很难甚至无法通过实验获得。那么，我们该如何对这类问题提出看似合理的答案，又该如何判定这些答案的证据是否足够强呢？（这个主题属于 *FRISCO* 中的 I，即推理（inference），因为我们讨论的是**推断至最佳解释**。）

在本章中，你首先会看到一种处理非实验假设的方法，它与你在上一章中看到的处理实验假设的方法非常相似。其基本准则和推理逻辑是相同的：粗略地说，假设的支持来自于它们解释事实的能力，以及其竞争对手缺乏这种能力。但不同情境下的证据类型差异很大，因此对探究领域的深入了解也是必不可少的。

由于因果关系与这里讨论的解释类型至少有着密切的联系，因此本章专门设有一节讨论因果关系。此外，由于许多人需要将他们的假设及评估付诸笔端，本章还专门设有一节讨论写作。

不同类型的论证有不同的评价准则。既然你现在已经熟悉了两类论证（演绎论证和最佳解释论证），并且在接下来的两章中还会遇到另外两类，那么问题就来了：“你应该应用哪些准则？”在本章中，你也将找到处理这一问题的建议。

首先，让我澄清一下我所假定的**假设**（hypothesis）与**结论**（conclusion）之间的区别。**假设**是一个命题，它的支持来自于它解释事实的能力以及其竞争对手无法解释事实的能力，并且它是该情境下待检验的候选方案。称某事为假设意味着我们尚未判定它是正确的。当某人根据证据判定它是正确的，那么对他而言，它就成了一个**结论**。此时，试探性的性质已经消失。此外，还存在一些中间情况，例如我们可以称某事为“初步结论”。

然而，并非所有结论最初都是假设。例如，有些结论提出价值主张（如“每个物种的栖息地都应该得到保护”、“我们应该喷洒双硫磷来杀死蚊子幼虫”以及“阿琳做法是错误的”），这些结论并不解释任何事情。（但是，假设某人相信其中一个价值命题，这可能会解释该人所做的某些事情。例如，假设法官认为阿琳是错误的，这可能会解释他给她的判决为何如此严厉。）这些价值主张不是假设，因为它们的支持并非来自于它们解释事实的能力，也不是为了解释事实而提出的。它们通过其他方式获得支持，这将在第 11 章中讨论。

此后，为了避免不得不同时说“假设”和“结论”这两个词，我通常只使用**假设**一词。但请将我一般性的论述也视为适用于那些曾经是或可能是假设的结论。不过，当谈到在特定情境下显然是结论的事物时，我会使用**结论**一词。

### 提出最佳解释假设

面对本章开头提到的那些问题时，如果对相关课题有充分的了解，将会大有裨益，这样我们就能避免徒劳的想法，并且以往类似情境中的假设可能会激发当前情境下的好想法。通常，对问题进行苦思冥想也有帮助，但有时暂时跳出问题也是个好主意。与他人交流、想象（至少暂时地）我们所处的一些限制是不存在的，以及对情境中存在的暗示保持敏感，这些都有帮助。通常，以应对困难的方式对不成功的假设进行修改和完善，然后进一步完善，使得假设的产生和测试相互作用、相互促进，这样做是很有帮助的。但是，并没有一套通用的规则，只要遵循了就能产生好的假设。其中很大一部分是直觉性的、个人化的，并且依赖于我们对该课题已有的大量了解。

假设我们有了一个由我们或他人提出的临时假设，我们该如何判断它呢？现在我们转向这个主题。

### 应用接受最佳解释假设所需满足的准则

评判最佳解释假设的准则与评判实验假设的准则相同，实际上实验假设就是最佳解释假设的一种。也就是说，无论我们是否进行了实验，一个假设的合理性取决于：在合理的假设前提下，(1) 它解释了一些事实；(2) 事实的其他替代解释与某些事实不一致（如果存在一个看似合理的替代假设，那么我们绝不能支持该假设）；(3) 它本身与事实一致；以及 (4) 它是看似合理的（plausible）。区别在于，当我们无法通过实验来创造证据（*FRISCO* 中的 *R*，即现实情况/研究）时，我们必须像侦探一样去搜寻证据。采用这些准则的推理（*I*）（无论是否是实验性的）被称为**最佳解释推理**。这里使用“最佳解释”一词，是因为假设的可接受性取决于它是对事实的最佳解释（同时也是对事实的一个良好的解释）。

让我们将这四个最佳解释准则应用于法医的结论——那一刀的力度是中等的：

1. 该结论解释了为什么伤口只有 2.5 英寸深。

2. 另一种深度的解释是那一刀的力度很大，但撞到了骨头。在合理假设前提下，这种替代解释与“伤口周围骨骼没有痕迹”的观察结果不一致。另一种替代解释是那一刀力度很大，但刀被艾尔（Al）佩戴的奖章挡慢了。考虑到合理的假设是，如果当时有奖章，肯定会有人报告，而这种替代解释与“检查过艾尔的人都没有报告奖章”这一事实不一致。所有其他看似合理的替代解释似乎也都与事实不一致。

3. 在合理假设前提下，该结论似乎不与任何事实不一致。如果我们假设艾尔穿着一件非常厚的带饰钉皮夹克，该结论就会与观察到的深度不一致。（在这种假设下，中等力度的刀伤甚至无法刺入 2.5 英寸。）但这种假设似乎是不合理的。没有人提到过这种夹克，在艾尔躺在餐厅地板上拍摄的照片中也没有这种夹克——只有一件薄尼龙夹克。

4. 该结论是看似合理的——至少与任何替代方案一样合理。

因此，法医关于那一刀力度中等的结论根据这些准则似乎是合理的。如你所见，这些准则并不局限于实验推理。它们适用于许多其他类型的假设，包括对本章开头问题的可能回答，这些假设可以被描述为历史假设、对作者含义和意图的解释，以及关于他人心理状态（信念、喜爱、意图、厌恶、恐惧等）的假设——或者广义上，关于不可观察或难以观察的事件或事态的假设。

### 拒绝最佳解释假设的推理模式

拒绝非实验性假设的推理模式与实验推理相同。第一阶段是表明，假设 (*h*) 加上假设前提 (*a*) 的一个推论 (*i*) 如果为假，则需要否定假设和假设前提的组合。例如，那一刀力度很大（称之为 *KS*，代表 *knife* 和 *strong*）的假设，连同“那一击没有障碍物”的假设前提以及“无障碍的高力度刀伤会刺入超过 2.5 英寸”的假设前提（*NOUP*），暗示了伤口会超过 2.5 英寸深（*WM*）。因为它没有超过 2.5 英寸深，所以假设和假设前提的结合必定为假，如例 9:1 中的否定后件推理所示，它描述了推理模式的第一阶段。

#### 例 9:1

同一种推理的两种替代符号表示：

| | |
|---|---|
| 1. 如果 *h* 和 *a*，那么 *i*。 | 1. 如果 KS 且 NOUP，那么 WM。 |
| 2. 非 *i*。 | 2. 非 WM。 |
| 3. 因此，并非 *h* 和 *a* 同时成立。 | 3. 因此，并非 KS 且 NOUP 同时成立。 |
| 或：非 (*h* 和 *a*) | 或：非 (KS 且 NOUP) |

否定后件（Denial of the consequent）；演绎有效。

左侧的符号表示中使用了示意字母 *h*、*a* 和 *i*（分别代表假设、假设前提和推论）。左侧符号表示中使用字母 *i* 代表推论（而不是用 *p* 代表预测），是因为“预测”一词仅限于对未来的预期。除此之外，这种模式与前一章关于实验的模式相同。这里呈现的模式比实验预测的模式更具普遍性，但也包含了后者。

例 9:1 右侧的符号表示中使用了代表内容的字母 *KS*、*NOUP* 和 *WM*。我也可以使用 *p*、*q* 和 *r*。你应该使用任何对你最方便的字母系统。

第二阶段是表明，肯定假设前提随后需要否定该假设。我们通过声称这些假设前提为真来实现这一点。在这种情况下，假设前提是那一击没有障碍物，以及无障碍的高力度刀伤会刺入超过 2.5 英寸。这种模式如例 9:2 所示。

#### 例 9:2

| | |
|---|---|
| 4. 并非 *h* 和 *a* 同时成立 | 4. 并非 KS 且 NOUP 同时成立 |
| 5. *a* | 5. NOUP |
| 6. 因此，非 *h* | 6. 因此，非 KS |

肯定否定联言命题的一个部分（Affirming one member of a negated conjunction）；演绎有效。

复习一下：假设那一刀力度**很大**。其中一个假设前提是那一刀没有受到阻碍。另一个是无障碍的高力度刀伤会刺入超过 2.5 英寸。为了暂时简化，我们只考虑这些假设前提（尽管还有更多）。如果该假设和假设前提为真，那么阿琳那一刀就会刺入超过 2.5 英寸。（这个条件命题是例 9:1 中的第 1 行。）但那一刀刺入没有超过 2.5 英寸（第 2 行），所以并非假设和假设前提同时为真（例 9:1 中的第 3 行，在例 9:2 中重复并标为第 4 行）。我接受这些假设前提（例 9:2 中的第 5 行）。所以我拒绝那一刀力度很大的假设（例 9:2 中的第 6 行）。

请注意，拒绝替代假设取决于对假设前提的接受。例如，如果关于那一击没有障碍物的假设前提是不可接受的，那么该假设并未被这种推理反驳。不要总是将**错误**的推论归咎于假设！

与实验推理一样，这种推理模式在某种程度上是理想化的。在实践中，我们通常需要通过添加诸如“可能”、“倾向于”和“通常”之类的词来稍微放松其严密性。例如，我提到的其中一个假设前提可能应该包含“可能”这个词：“无障碍的高力度刀伤**可能**会刺入超过 2.5 英寸。”相应地，对假设的拒绝应读作：“那一刀力度**可能**并不大。”将演绎推理这样放松并没有机械性的规则。正如第 7 章在“宽泛推导”标题下讨论的那样，一个人必须运用自己的良好判断。

总之，我们在拒绝假设时使用的准则本质上就是宽泛推导的准则（在第 7 章中讨论）。这些准则的使用属于 **FRISCO** 中的 *I*。

### 细节与常识

我只对这个最佳解释推理示例进行了简略描述。如果没有对情境细节的更完整掌握，并且没有关于此类情境的常识，你将无法根据给出的事实判断法医的结论是否真正合理。例如，那把刀可能非常钝。如果是这样，那一刀力度很大的替代结论将与证据一致，并能很好地解释伤口的深度。因此，对细节的了解很重要。（提醒：如果存在一个看似合理的替代假设，就不应支持所讨论的假设。）

此外，据我所知，成年男性的胸骨和软骨可能非常紧密且坚固，以至于薄钢片如果不费大力就无法刺入。如果是这样，有力的一击解释将是合理的。排除这种替代方案需要应用到该情境的常识。法医应该拥有这种知识。我们陪审员并没有这种知识。（由于专家和陪审员之间的这种差异，专家被允许在法庭作证中陈述他们的结论，你会在第 4 章中记得这一点。）

这里的一般教训是：如果你进行最佳解释推理或评判他人的最佳解释推理，你必须熟悉情境的细节，并具备关于情境中事物的常识。有时常识来自为了学校教学而组织的标准学科（例如化学），但往往并非如此，比如力度中等那个案例。这里的知识来自多个学科——加上来自实际经验的大量常识。

### 其他类型的解释

“解释”一词适用于各种事物。在假设检验中，它适用于解释某事的命题或因素。[^1] 但还有其他类型的解释，包括过程解释（例如解释如何游泳）、阐释解释（例如解释“演绎有效性”的含义）以及辩护解释（例如解释为什么开车时不打信号灯就不应变道）。相比之下，最佳解释推理中的解释应该是解释过去或现在的事实。它们的解释不是描述一个过程、陈述一个含义或为某事辩护。我提到这些其他类型的解释是为了帮助你避免将这些解释与假设或结论的解释混淆。在寻找解释某事的理由时，不要寻找这些其他类型的解释（尽管这些其他类型的解释可以与解释某事的尝试相关）。

### 解释与部分解释

在力度中等案中，被解释的“某事”是伤口只有 2.5 英寸深这一事实。试图解释这一事实的命题是“那一刀力度只是中等的”。该命题是法医的结论。它解释了为什么伤口只有 2.5 英寸深。

[^1]: 它也适用于*声称*该命题或因素解释了某事发生的原因，或解释了，或引起了某事。例如，以下可能是假设：“那一刀力度的中等*解释了为什么*伤口只有 2.5 英寸深”、“那一刀力度的中等*解释了*伤口只有 2.5 英寸深的原因”以及“那一刀力度的中等*引起了*伤口只有 2.5 英寸深”。为了简化讨论，我将不详细讨论这种本身包含“引起了”、“解释了”和“解释了为什么”等术语的假设形式。我邀请那些对此感兴趣的人对这种假设进行扩展。
因素和命题一样，都可以解释事物。如果用因素而不是命题来看待力度中等案，那么该因素就是那一刀的（所谓的）中等力度。当提供一个因素作为解释时，那么该因素获得、到位或作为情境特征（虽然通常很生硬）的命题就可以是假设。例如：“刀伤力度的中等是该情境的一个特征。”更地道地，我们会说：“那一刀是中等力度的。”假设总是以完整句子形式表述的命题。因素不是以完整句子形式表述的。在讨论命题和讨论因素之间的区别通常只是表达方便的问题。

#### 部分解释

最佳解释结论不需要是事实的全部解释。相反，它可以是解释的一部分，或部分解释（partial explanation）。其区别往往在于我们看待事物的方式。例如，在给定条件下，我们可以将那一刀的力度中等视为**该**解释。或者我们可以将其视为伤口深度解释的**一部分**，刀刃的锋利程度也是一部分（锋利的刀阻力较小）。另一种说法是：刀的锋利程度可能被认为仅仅是一个条件，或者可能被认为是对结果解释的一部分。鉴于一个因素确实发挥了作用，我们会将其分类为一个条件、部分解释还是**该**解释，取决于为了分配责任而似乎正确强调的内容。无论是解释还是部分解释，力度中等的结论都从其解释或帮助解释事实（即解释或帮助解释事实）的能力中获得支持。

如果你对在某些情况下寻找**该**解释感到困扰，你可以转而将自己视为正在寻找一个连同某些其他情况一起解释这些事实的东西。在我们的案例中，你可能认为我们关注的是那一刀的中等力度，作为伤口深度解释的一部分，另一部分是刀的锋利程度。但由于刀的锋利程度是给定的，而那一刀的力度至关重要，我们对后者更感兴趣。

你可能已经猜到，重要的条件、提供部分解释的因素和提供全部解释的因素之间的区别是很微妙的。需要良好的判断，关注特定情境下的细节和目标。这取决于你觉得在该情境下有理由为该因素分配多少责任。

#### + 解释、说明与因果关系

这种解释（accounting-for）关系是什么样的关系？特别地，它是一种因果关系吗？这是一个困难且有争议的问题。许多人不愿意这么说，因为在他们看来，因果关系是一种僵化的、机械的连接。另一方面，如果因果关系被视为一个更广泛的概念，关注的是带来（bringing about）某事，那么这种解释关系被视为因果关系就更合理了。这样它将包含机械类型的因果关系，但也包含许多其他关系，包括那些关于动机的关系以及涉及意图和信念的关系。例如，在因果关系的广义视角下，以下所有都将是因果关系：“阿琳刺伤的动机是嫉妒”、“她的嫉妒解释了她刺伤艾尔的行为”、“她的嫉妒带来了她刺伤艾尔的行为”以及“她的嫉妒引起了她刺伤艾尔的行为”。此外，所有这些都会被认为在说类似的事情。在这种因果关系的广义视角（实际上是我的视角）中，将解释视为一种因果关系，使我们能够从本章稍后讨论的一些引导因果判断的原则中，获得在对可能的解释关系进行判断时的指导。

在更狭窄、机械的因果关系视角中，并非所有关于解释的判断都被分析为因果关系。但这些判断仍然可以做出，假设也可以被评判，因为我们通常能分辨出一件事是否能解释另一件事，就像在广义视角中，我们能分辨出一件事是否能引起另一件事一样。例如，在审判情境中，很明显阿琳的嫉妒可以**解释**她杀害艾尔的行为，就像（在广义视角中）很明显她的嫉妒可能**引起**她刺伤他的行为一样。

因为这是一个有争议的话题，你必须自己决定。但即使你将因果关系理解为机械性的，你仍然通常可以对解释关系做出明智的决定。试一试看看。

### 应用于力度中等结论的 FRISCO 方法

在以力度中等论证作为最佳解释推理的说明时，我已经使用了 FRISCO 方法的要素，通常没有明说。接下来，考虑将该方法显式应用于该示例：

**核心焦点（Focus）：** 法医的结论是那一刀的力度只是中等的。

**理由（Reasons）：** 她的一些更重要的理由如下：

1. 伤口深度只有 2.5 英寸。
2. 骨头上没有痕迹。
3. 刀刃锋利。
4. 没有证据表明该刀伤存在外部阻碍（如奖章或厚夹克）。

理由在陪审员看来都是可以接受的。前两个我们接受了法医的说法，她在可信性准则方面表现良好。我们不需要听信她关于第三个理由的说法，因为我们能够亲自检查那把刀。封存该刀具作为证据的侦探向我们保证，在他当晚捡起该刀后没有对它做过任何处理。第四个理由似乎是可以接受的，因为照片显示没有外部阻碍的证据，而且没有人提到任何外部阻碍的迹象。如果有证据，检察官肯定会提到。

**推理（Inference）：** 该结论满足最佳解释准则：它解释了伤口的深度。主要的替代假设（高力度假设）与这一深度不一致，因为不存在外部阻碍。法医的结论与我们所知道的事实一致，并且看起来是合理的。

在看出假设解释了深度时，我们假设那一刀没有被厚饰钉夹克或奖章阻碍。在拒绝那一刀力度很大的假设时，我们做了同样的假设前提，并且还假设如果那一刀力度很大且没有阻碍，刺入深度会超过 2.5 英寸。第一个假设前提似乎是可以接受的，理由是如果存在障碍物，检察官会提到它。对于最后一个假设前提，我们必须依赖于我们自己的经验和法医的可信性，如果假设前提为假，她可能就不会得出那个结论。虽然还有其他假设前提，但这些似乎是最重要的。

**情境（Situation）：** 在确定最受关注的替代假设时考虑了情境。如果那一刀力度很大，那么阿琳就会知道她那一击可能会造成严重的身体伤害。如果她知道这一点，那么她可能犯有谋杀罪。（第 7 章中的检查练习项 7:8 提供了谋杀指控的细节，如果你还不知道，它将使你明白为什么会这样。）因此，高力度替代方案是重要的。在注意到没有人提到该刀伤存在外部阻碍时也考虑了情境。我必须了解情境才能意识到这一点。此外，情境使得“中等”（moderate）一词的模糊性变得可以接受（这一点将在“清晰性”下详述）。根据我对情境的理解，它支持法医的结论。

**清晰性（Clarity）：** 在这种情境下，术语的含义似乎足够清晰，而且它们似乎没有在多个意义上被使用。特别是，“中等”一词虽然模糊，但在该情境下对陪审员的目的来说已经足够精确：基本上，我们必须决定控方是否已经排除合理怀疑地证明被告知道她的行为造成了严重身体伤害的高度可能性，这是谋杀指控的关键特征。基于法医的证词和关于那一击是中等的论证，我们判定谋杀准则的这一关键特征未得到满足，并据此认定阿琳不犯谋杀罪。“中等”一词在那种情境下清晰到足以让我们得出结论。（顺便提一下，请注意 FRISCO 各部分之间的相互依赖性，特别是 *I、C* 和 *S* 之间，这是常见现象。）

在我们审议的这一点上，“排除合理怀疑地证明”这一短语的含义没有受到质疑，尽管如果有人认为已经排除合理怀疑地证明阿琳知道她那一击造成了严重身体伤害的高度可能性，那么它可能就会受到质疑。对于这种情况，这个短语足够清晰。后来，在考虑阿琳使用她所用的力度是否合理时，这个问题被提出了，因为其中一名抵制阿琳犯有故意杀人罪（voluntary manslaughter）判决的陪审员似乎在使用演绎的证明概念。这种情况在第 12 章中有更详细的讨论。

**综述（Overview）：** 在复查时，我注意到可信性准则也直接适用于该结论，因为法医在证人席上陈述了结论。法医在这些准则上表现良好：她有背景经验和知识，她没有明显的利益冲突，没有人不同意她的看法，她有良好的声誉，我不知道既定程序（可能的一个弱点），她知道自己名誉受损的风险，她给出了理由，而且我没有理由认为她没有细心的习惯。

在审查了证据、情境以及我应用各种准则的方式后，我坚持相信法医假设的决定。请意识到，许多细节已被省略——这在所难免。但叙述确实包含了主要关注点，并确实说明了 FRISCO 方法在涉及非实验性推断至最佳解释的真实情境中的应用。

这种力度案例将可信性准则的应用与最佳解释准则结合起来，是广泛案例中的典型代表。通常我们了解得足够多，可以在一定程度上应用最佳解释准则，但了解得还不够多，无法完全自信地应用它们，在这种情况下，我们需要对来源的可信性做出判断，为此我们需要可信性准则。

### 在弄清某人的信念或想法时的最佳解释推理

当我们得出某人相信或思考某事的结论时，我们的结论是一个最佳解释结论。通往结论的推理取决于其最佳解释事实的能力。例如，考虑佩德罗（Pedro）的结论：吉姆（Jim）相信玛蒂娜昨晚和卡尔在“蓝云”（Blue Cloud）约会。这个结论将解释为什么吉姆说他看到他们在一起。它将解释为什么吉姆跑到佩德罗那里告诉他这个消息（假设吉姆认为佩德罗爱着玛蒂娜，因此会非常感兴趣）。它将解释为什么吉姆第二天早上避开卡尔，即使他需要卡尔帮他做作业（吉姆和佩德罗是死党）。另一个替代假设“吉姆试图让佩德罗心烦意乱，以便在壁球比赛中更容易击败佩德罗”似乎与佩德罗认为的吉姆的性格不符。你能在这里看到最佳解释模式吗？

对这种模式的一个更复杂的说明可以在莎士比亚评论家 A. C. 布拉德利（A. C. Bradley）支持他对伊阿古（《奥赛罗》中的反派）的妻子爱米利娅（Emilia）的部分理解的论证中找到。你在第 7 章的一个演绎示例中看到了布拉德利这部分解释的使用，其结论是伊阿古不是一个通俗剧式的反派。布拉德利得出该结论的理由之一是他关于爱米利娅信念的（中间）结论，即她不怀疑伊阿古是恶棍。布拉德利支持这一中间结论的论证出现在这里，为了方便参考，对句子进行了编号。据我理解，第 1 句是这段文字的结论。

**例 9:3[^2]**

(1) 她从未梦想到他是个恶棍。…… (2) 她在看到奥赛罗为那块手帕心神不宁时，未能产生任何关于阴谋的怀疑，这表明她对丈夫是多么深信不疑。(3) 甚至她谈话的语气，以及她提到那个（她认为）挑起伊阿古对她产生嫉妒心的无赖，都无可置疑地证明，伊阿古就是那个坏蛋的想法从未在她的脑海中闪过（IV.ii.115–147）。(4) 如果对此还有任何犹豫，那么在听到奥赛罗说“你丈夫知道这一切”后，她那连喊三声充满惊讶和恐惧的“我丈夫！”，肯定会打消这些犹豫；(5) 还有当伊阿古进来时，我们在她的申诉中听到的那种哽咽的愤慨和绝望的希望：

> 如果你是个男人，就戳穿这个恶棍  
> 他说你告诉他他的妻子不忠  
> 我知道你没有，你不是那样的恶棍  
> 说话吧，我的心都要碎了。[^3]

请记住，奥赛罗刚刚因误解的嫉妒杀死了他的妻子。

[^2]: 感谢布鲁斯·华纳（Bruce Warner）让我注意到这个例子。

布拉德利在这里极其自信的语气可能暗示他提供了一个演绎论证，但就目前而言，结论肯定不是必然得出的。通过添加适当的假设前提，我们可以将该论证转换成一组演绎有效的论证，每个论证都能得出结论，但我们似乎需要添加以使论证演绎有效的假设前提（至少是我能想到的那些）也是虚假的，这使得论证变得软弱。

例如，请考虑例 9:4 中的一个论证，它利用了她那“连喊三声充满惊讶和恐惧的‘我丈夫！’”，其中包括我添加的一个可能假设前提（*b*）。

#### 例 9:4

**a.** 爱米利娅连喊三声充满惊讶和恐惧的“我丈夫！”[^4]

**b.** [可能的假设前提：如果一个女人在得知丈夫做了一件邪恶的事时，连喊三声充满惊讶和恐惧的“我丈夫！”，那么她之前并不怀疑他是恶棍。]

**c.** 爱米利娅刚刚被告知她的丈夫做了一件邪恶的事。

**d.** 因此，爱米利娅并不怀疑她的丈夫是恶棍。

你会记得，这个论证的形式是肯定前件。这是一种演绎有效的形式。在本案中，前件在理由 *a* 和 *c* 中得到肯定。整个条件命题是 *b* 中给出的假设前提。

例 9:4 中论证的问题在于，条件命题（*b*）似乎是虚假的。在这种条件下，女人可以伪造惊讶和恐惧的语气。此外，即使她确实怀疑丈夫是恶棍，她仍可能被行为的残暴程度所震惊和恐惧——只是没想到会恶毒到这种程度。或者，即使她认为他是个可怕的恶棍，但不认为他的恶行会朝这个方向发展，她仍可能感到震惊和恐惧。或者，即使他的这种恶毒是她知道的，她仍然可能对他恶毒的行为惊愕和恐惧。我对恐怖分子所做的一些可怕的事情感到惊愕和恐惧，尽管我事先知道有恐怖分子，他们做可怕的事情。

[^3]: A. C. 布拉德利，*莎士比亚悲剧*（伦敦：麦克米伦有限公司，1937年），第 179-180 页。
[^4]: 你可能已经注意到，我增加了“……的语气”，以便明确我们可以观察到的内容。如果你想照搬布拉德利的陈述，那么你必须重做这个例子，使最佳解释推理从观察到的结果推导到她感到惊讶和恐惧的结论。

布拉德利暗示他有一个无可置疑的证明。我觉得他夸大了事实。至少正如我所指出的，他没有一个理由为真的演绎有效论证。对他的论证进行同情性理解（sympathetic interpretation）会采用最佳解释推理的标准。她从未梦想到他是个恶棍的结论可以解释她连喊三声充满惊讶和恐惧的“我丈夫！”。这也可以解释她明显的哽咽愤慨、她在手帕事件中显然没有产生阴谋怀疑，以及布拉德利提到的其他谈话的语气。所有这些证据也往往使“她确实怀疑他是恶棍”的替代假设显得不可信。作为一个最佳解释论证，证据可以堆叠起来，但在这里，没有哪一条证据是结论性的。

像往常一样，还有很多话要说。这个案例的一个有趣特征是爱米利娅是一个虚构人物。因此，整个论证和解释都是**假装**她是一个真实的人来进行的。

另一个有趣的特征是布拉德利极度的自信。如果你深入阅读和讨论这部剧，你可能也会像我一样认为布拉德利极度的自信是没有根据的。但我只是想向你展示一种看待我们所做的许多推理的方法，而你很可能判断他的自信是有根据的。在我看过的《奥赛罗》演出中，伊阿古并没有被诠释为一个通俗剧式的反派。布拉德利的观点显然已被普遍接受。

### 摘要

当我们无法使用实验时，就像我们面对各种假设时一样，我们仍然可以使用最佳解释推理模式。对假设的两阶段拒绝取决于其他假设前提。对假设的接受取决于排除其他替代方案的方法（使用与实验相同的四个准则），并且也取决于假设前提。这两个过程都需要熟悉情境的细节并具备所研究事物的相关知识。通常必须通过使用诸如“可能”和“通常”之类的词来放松推理。

使用 FRISCO 方法可以方便地评估最佳解释结论。这不是一个机械化的过程。这些要素是相互依赖的，不一定按照陈述的顺序应用。它们按照任何适合你对情境日益掌握的顺序来应用，并经常重新应用。

以下是最佳解释推理的基本策略：深入了解该主题或研究领域。在合理假设前提下，寻找似乎是最佳解释的假设。与他人讨论。对你认为情境中存在的限制条件保持开放态度，可能是暂时性的放松。寻找替代假设。寻找与各种假设（包括受青睐的假设）不一致的证据。如果在彻底搜索反面证据后仅剩下一个假设，那么——或许是非常试探性地——接受它。如果你付出了巨大努力去揭示替代假设，并且它是唯一可能的解释，而所有其他解释都显然不可接受，那么接受它。但要对可能使该假设变得可疑的新证据保持开放。如果你发现了一些，请考虑完善该假设，但有时最好直接放弃它。

我们的许多推理都是最佳解释推理。最佳解释推理适用于历史假设、“侦探推理”（who-done-it）假设、关于人们心理状态、信念和意图的假设，以及其他各种关于我们无法（或不易）直接体验的事物的假设。最佳解释推理不适用于价值结论，尽管最佳解释结论通常与价值结论相关。

这里关注的解释类型是旨在说明某事的解释。其他类型的解释包括过程解释（解释如何做）、阐释性解释（解释术语的含义）和辩护（解释为什么某事应该被相信）。尽管这些解释通常涉及说明性解释，但它们是不同的，并且通常具有不同的成功准则。

&#43; 最佳解释推理中这种说明性（accounting-for）关系的本质是复杂且有争议的。我的观点是，这种说明性关系是一种因果关系，将因果关系广义地理解为一种带来（bringing-about）的关系，包括机械性的以及其他类型的带来某事。但有些人觉得“引起”（cause）这个词表示的是一种机械关系。如果你有这种感觉，那么只需坚持将“说明”作为你对本章讨论的解释类型的理解。它会为你提供很好的服务。

## 必然推出：演绎推理与最佳解释推理及一般论证策略

在最佳解释推理中，结论并不必然从证据中推出。如果它必然推出，那么我们就可以直接根据证据推导出结论（演绎结论），而无需担心结论的解释力。最佳解释推理中的策略基本上是一个表明一个可能的结论可以解释证据并消除其他可能的解释（因为它们不可信），从而只留下一个的过程。但结论并不必然从证据中推出，因为其中一个假设可能是错误的，而且可能存在另一个我们尚未想到的看似合理的解释。

从伤口的深度和骨头上没有痕迹并不能必然推出那一击是中度的。病理学家提到的这些理由并不能排除例如受害者佩戴了奖牌而使那一击偏转并减弱力度的可能性。这个替代假设没有引起陪审员的注意。如果它引起了，它可能会因为以下理由而被拒绝：如果艾尔佩戴了奖牌，那么它的存在可能会在审判中被提及，而它的存在并没有被提及。无论如何，我们无法绝对确定没有其他可能性，所以结论并不必然从她所说的内容中推出。

**a.** 刀伤造成的伤口只有2.5英寸深。

**b.** 附近的胸骨上没有痕迹。

**c.** [如果一个刀伤造成的伤口只有2.5英寸深，且附近的胸骨上没有痕迹，那么那一击的力度只是中等的。]

**d.** 因此，那一击的力度只是中等的。

通过添加一个适当的理由（*c*），我们将论证转化为一个演绎有效的论证。但这并不能使论证成为一个好的论证，因为添加的理由不是真的。奖牌的可能性表明了这一点。

你可能从例 9:5（以及从你在第7章项目中的经验）看出，你可以通过添加适当类型的理由，将任何论证转化为演绎有效的论证。然而，经常会产生一个糟糕的论证，因为论证随后包含了一个虚假的理由。演绎有效性对论证毫无帮助。最佳解释往往是最好的方法。

### 在最佳解释方法与演绎及其他方法之间选择

通常，在评判一个论证时，我们的目的是决定结论是否为真或是否得到充分确立。假设这是我们的目的。因为有各种看待论证的方式（演绎的、宽泛演绎的、最佳解释的、概括的和价值判断的——后两者尚未在本书中讨论），我们应该选择那种能为结论提供最强支持的看待论证的方式。如果即使以使其看起来最强的方式看待论证，仍然没有提供足够的支持，那么它就不成功。然而，如果以某种方式看待它确实提供了足够的支持，那么它就成功了。

如果我们能够通过提供真实理由（以假设的形式）使一个论证成为演绎有效的论证，而且原始理由是真实的，那么我们应该采取这条路线并接受论证和结论。那么我们就会表明结论必然从真实的理由中推出，从而表明结论为真。还有什么比这更好的呢？

但通常我们能想到的任何产生演绎有效性的理由集合都包含一个虚假的理由。那么最佳解释方法可能是要使用的方法。它可以产生得到充分确立的结论，就像它对大多数科学结论所做的那样。一个熟悉的得到充分确立的最佳解释结论的例子是检察官的结论，即阿琳杀死了艾尔。参见第2章开头检察官提出的支持这一结论的复杂最佳解释论证。最佳解释结论有时得到非常强的支持，即使它们并不必然从证据或真实假设中推出。检察官的结论得到非常充分的确立。它得到了排除合理怀疑的证明，这是你能得到的最佳解释结论的最好的证明，但结论并不必然从证据中推出。

另一个熟悉的例子是病理学家关于那一刀力度中等的结论的论证。这也是最佳解释推理，尽管它没有像排除合理怀疑的证明那样强。实际上，在日常生活中，最佳解释推理通常不会产生排除合理怀疑的证明。但我们仍然经常有理由以此为基础做出决定。

通常，宽泛推导（见第7章）是我们能找到或构建的支持结论的最好的论证类型。即使结论并不必然推出，论证通常也可以是好的，但这里除了应用适当的准则外，还需要良好的判断。

价值判断论证不能期望符合最佳解释标准，因为价值结论并不说明事实。所以不同的策略（通常是宽泛推导）在这里是适当的。但一般思想仍然成立：为一个论证尽力而为（假设目标是决定结论是否可接受），然后仔细评判它。

### *复杂的组合*

通常复杂的论证由满足不同标准的部分组成：可能有些部分满足演绎（或宽泛推导）标准，有些部分满足最佳解释标准，有些部分满足概括标准（将在第10章讨论），有些部分满足价值判断标准（将在第11章讨论）。在阿琳的审判中，她杀死了艾尔的结论是最佳解释结论。但她犯了过失杀人罪的结论必然满足过失杀人罪的四个条件。第7章中例 7:1 出现了过失杀人罪的完整条件集。如果你没有看过这些条件，请现在回去看看。

这些条件中的第一个，粗略地说，是她杀死了艾尔。这是最佳解释结论（检察官的论证在第2章中提出）。她使用武力是不正当的这一结论（第四个条件）是价值结论。第二和第三个条件，我在这里为了简化起见略去了，需要最佳解释和价值判断论证。所以支持她在过失杀人罪指控下有罪的总体论证使用了至少三种不同类型的推理：最佳解释推理、价值判断推理和演绎推理。演绎推理是从满足四个条件到她是有罪的结论。

### 并非必然

一种经常听到的对论证的回应类型是"并非必然"或"那并不必然推出"。这在讨论中有时是一种误导性的举动，所以要当心。

通常，现实生活中论证的结论并不必然推出，所以"并非必然"的抱怨通常是正确的，尽管有时具有误导性。但危险在于这些词会被应用于一个好的最佳解释论证，或某种其他好的非必然论证，我们会认为论证不知何故被表明是糟糕的论证。

例如，一位陪审员指出，阿琳杀死艾尔的结论并不必然从证据中推出。从字面上看，这是正确的，尽管具有误导性。也就是说，如果我们随后认为检察官的论证是一个糟糕的论证，那就是误导性的。

结论必须必然推出的要求往往是错误的准则，即使在某人的生命和自由岌岌可危的法庭案件中也是如此。那里的准则是*排除合理怀疑的证明*，一个适当的准则，但不如*必然推出*的准则严格。检察官确实排除合理怀疑地证明了阿琳杀死了艾尔，尽管这个结论并不必然从证据中推出。

有时当人们说"并非必然"时，他们的意思是论证不是一个强有力的论证，并不是真的只意味着"并非必然"。另一方面，有时他们是字面意思。如果可能的话，要做的是找出他们的意思是什么，并根据他们的意思（**FRISCO** 中的 C）来处理指责。

如果一个人是字面意思，并将这一指责应用于最佳解释论证，一种适当的回应是："没错，但我并没有说结论必然推出。"一种更具攻击性的回应是："没错，但那又怎样？"

*并非必然*有时是对论证的不公平攻击。要为此做好准备。

### 摘要

任何论证都可以通过添加一个或多个适当理由作为假设转化为演绎有效的论证，但结果往往是一个糟糕的论证，因为一个或多个添加的理由是虚假的。当结论是事实性结论而非价值结论时，也就是说，是一个可以说明事实的结论时，最佳解释推理往往是一个很好的替代方案。

评判论证的一个合理策略是首先检查它是否满足演绎标准且其理由为真。如果是，那么接受它和结论。如果不是，那么应用归纳标准（本章和前一章讨论的最佳解释标准，或第10章讨论的概括标准），或价值判断标准（第11章讨论），或宽泛演绎标准（第7章讨论）。在给予论证一个公平的机会之前不要拒绝它。许多论证是强有力的论证，即使结论并不必然从给出的理由中推出。

通常一个论证有不同的部分，对不同部分需要不同的标准。但总体论证不会比其最弱的必要环节更强。

有时*并非必然*是一种谴责论证结论的欺骗性方式。在判断它是否在该情况下成立之前，要清楚这个短语在其语境中的含义。

## + 因果关系

大多数（如果不是全部）最佳解释推理都涉及因果关系。但由于因果关系是一个困难且有争议的话题，我把它留在这两章关于实验和最佳解释推理的末尾。因为因果推理符合最佳解释推理的模式，这里的一些想法已经提出过，但这里的重点是因果关系。

由于本节关于因果关系比本书其他部分更理论化，匆忙的读者可能有理由选择省略它。但对于深入处理许多领域的因果问题，本节可能有用。

### 单称因果陈述与因果概括的区别

当病理学家断言"刺伤导致了艾尔的死亡"时，她是在做一个*单称因果陈述*。做一个单称因果陈述就是说一个特定的事物（或事物集合）导致了另一个特定的事物（或事物集合）。一个单称因果陈述是关于一个序列的。虽然它可能是一个非常复杂的序列，但它只是一个序列。刺伤-死亡序列就是一个序列。

另一方面，"刺伤导致人死亡"这一陈述是一个*因果概括*。它是关于刺伤和死亡的无限多个可能序列的，而不仅仅是一个。

### 因果概括

以下是更多因果概括的例子：

> 吸烟导致癌症。
> 饮酒导致事故。
> 吹口哨使人安静下来。
> 松开油门踏板使汽车减速。

因为这些因果概括涵盖了无限多个可能的序列，它们不可避免地涵盖了超过其证据的内容。

#### 模糊性

此外，这种概括中肯定实例的比例往往是模糊的。也就是说，它们甚至没有大致说明肯定实例的比例。例如，吸烟被声称导致癌症是偶尔、有时、经常、通常还是总是？概括没有说明。

因为因果概括在其普通裸形式（"*X* 导致 *Y*"或"*X*s 导致 *Y*s"）中是模糊的，决定是否相信它们（并在你自己心中形成它们）的第一步是澄清它们的含义（**FRISCO** 中的 C）。澄清有两个特点：意识到短语*在标准条件下*（或某种类似短语）总是隐含存在的，并确定概括中属于何种普遍性程度。

以下是表示普遍性程度的术语示例：*总是、几乎总是、通常、经常、有时*和*可以导致*。当我说饮酒导致事故时，我并不是说饮酒总是导致事故，或者仅仅说饮酒可以导致事故。我认为饮酒*有时*导致事故（在标准条件下）。当我说吹口哨使人安静下来时，我心中的想法是吹口哨*通常*使人安静下来（在标准条件下）。当我说松开油门踏板使汽车减速时，我的意思是松开油门踏板（在标准条件下）*总是*使汽车减速。我意识到许多普遍性程度限定是模糊的，短语*标准条件*也是如此，但因果概括通常是模糊的命题。

因为其固有的模糊性和通常未指明的普遍性程度，因果概括需要特别小心。我们必须牢记标准条件限定符，并确保普遍性程度对情况来说足够清楚。如果油门弹簧断裂，那么松开油门踏板就不会导致汽车减速，但这并不能表明概括是错误的。条件不是标准的。什么会表明概括是错误的，将是反复出现的情况：汽车运转正常，条件在其他方面也是标准的，但松开油门踏板并没有导致汽车减速。

假设这在旧金山丘陵地带极其丘陵的地方下坡时发生。我不会将这些发生视为反例，因为对我和我的想象中我说这话的对象（住在平坦的伊利诺伊州中部）来说，旧金山的陡坡不是标准的。*标准*的概念，尽管可能是模糊的，也是取决于情况的。

如果全世界都像旧金山一样多山，那么适当的因果概括很可能是："松开油门踏板（在标准条件下）*经常*使汽车减速。"或者，如果情况需要明确关注下坡因素，我们可能会在原始概括中添加另一个限定词，使多山的地方成为明确的例外："松开油门踏板（在标准条件下）总是使汽车减速，除了下坡时。"

因此，因果概括有一个通常未说明且不可避免地有些模糊的*在标准条件下*限定词，这取决于情况，而且往往也有一个模糊性（关于其普遍性程度），这可以在某种程度上澄清，也取决于情况。在形成因果概括、形成和评估支持和反对因果概括的论证，以及试图将这些概括应用于新案例时，我们必须牢记这些观点。我们还应该牢记情况的具体细节（**FRISCO** 中的 S）。

#### 导致，或带来

考虑我关于这本书的一般因果假设（在第8章讨论）："使用这本书学习批判性思维课程，往往能提高学生的批判性思维能力。"虽然我在那里没有使用*导致*这个词，但这个假设在我听来是一个因果假设，可重述如下："使用这本书的批判性思维课程，往往导致学生批判性思维能力的提高。"这应该被理解为包括*在标准条件下*的限定词，显然包含了有些模糊的术语*往往*，对我来说它的意思是*多半时候*但当然不是*总是*或*几乎总是*。

如果这种使用*导致*一词对假设的重述在你看来是可以接受的重述，那么你可能和我一样认为因果关系不一定是机械的。如果这种重述让你觉得它将联系变得太机械而使你感到困扰，那么你应该在本章及许多其他地方用短语*带来*代替*导致*一词。在我听来，*导致*与*带来*意思相同，所以*带来*——用中性的术语来说——是我打算在这里讨论的主题。

但是什么样的证据可以支持一个一般因果假设呢？许多类型的证据可能相关，但往往没有一种类型是结论性的。

#### 概括的实例

对关于这本书的*可以导致*概括的强有力支持，将是以下单称因果陈述得到充分确立的一个案例："这个使用这本书的批判性思维课程，在这个情况下导致了学生批判性思维能力的提高。"（在第8章，你读到了关于实验性确立这种单称因果陈述的策略和问题。）更多的实例会更牢固地确立*可以导致*概括，也许可以证明*有时*，甚至*经常*概括是正当的。

#### 代表性

这些实例必须是代表性的，或者合理地被相信为代表性的，针对群体、区域或情况中被隐含或明确指称的，才能做出更普遍类型的概括，如*通常*或*往往*概括。许多代表*X*导致*Y*的肯定实例支持相应比例的因果概括。也就是说，十个中有八个肯定的代表性实例支持"*X*s 往往导致 *Y*s"形式的概括。例如，假设在十个相当有代表性的情况中，有八个情况将班级规模从五十人减少到十五人，导致实验组比对照组学到更多的批判性思维。但在两个情况中没有。那么我们可能可以得出结论，将班级规模从五十人减少到十五人往往能提高批判性思维的学习。但这样做，实例必须是代表性的，这一点至关重要。

#### 反复关联的支持

在某些领域（如批判性思维书籍和课程的领域），很难完全确信地确立任何单独的单称因果主张（包括我一直在谈论的那个）。我们不能然后在一组分别得到充分确立的单称因果主张的基础上建立概括，因为这样的主张不存在。但概括仍然可以从自变量和因变量的反复关联中获得一些支持，例如反复关联使用这本书与学生批判性思维能力的提高。但这种证据本身并不充分，因为结果可能有另一种解释。例如，在所有这些实例中，学生可能带着学习批判性思维的强烈意图来到课程，尽管使用了这本书，还是学会了这样做。或者，学生可能只是变得更加成熟，或者在其他课程或经历中学会了批判性思维。反复关联是有趣的，但单独使用是危险的。气压计下降和随后风暴的出现是反复关联的一个例子，其中两个事件都不是由另一个引起的。

反复关联作为证据的一个重要特点是，原因必须发生在（或与）其结果同时）其效果之前。原因不能发生在其结果之后。

#### 刻意干预

如果自变量是由某人故意引入的，反复关联的价值可以得到加强。例如，我的朋友在发动机运转时故意多次转动钥匙，这立即伴随着一声巨响，支持了"在发动机运转时转动钥匙导致巨响"的概括，因为似乎没有其他合理的解释既能解释故意转动钥匙，又能解释每次都紧随其后发生的巨响。同样，反复关联使用这本书和批判性思维能力的提高所得到的支持，会因为实验者将这本书反复介绍给一半不是特别有兴趣学习批判性思维的学生群体，并发现使用这本书的学生比其他人更具批判性思维而得到加强。这种支持来自于难以提出替代假设来解释结果。

这种刻意干预的考虑是支持使用实验而非机会性数据收集的主要原因之一。当然，也有缺点。实验者经常打乱事物的自然流程，以至于结果不符合世界的真实情况。

证据不必来自*添加*一个因素。如果第一个变量是由研究者故意*减少*或*排除*的，而且两个变量之间的关系是有道理的，那么另一个变量的相应减少或消除就为因果概括提供了很好的支持。这种推理被用于将胆固醇摄入与动脉中的胆固醇涂层联系起来（这反过来在当时被认为会导致心脏病发作）。研究者减少了胆固醇摄入水平，发现它伴随着动脉胆固醇涂层的平均减少或消除，于是得出结论，第一个减少导致了第二个减少。

#### 因果链

独立变量和因变量之间的因果连接路径（或因果链）有道理（以满足假设是合理的这一标准）也很重要。我们知道从松开油门踏板通过关闭节气阀到减少进入发动机的空气和汽油到发动机减速的因果链，使连接链条两端的因果概括更加可信。在减少批判性思维课程班级规模的情况下，根据我的教学经验，我认为这个因果链条件是满足的。减少班级规模通常会导致每个人更多的讨论，从而更多的例子练习。它通常还会导致教师更多的个别关注和教师与学生之间更多的互动。这些事情反过来可能会导致批判性思维学习的更多提高。

同样，饮酒导致事故的因果概括得到了从饮酒通过延长反应时间、判断力下降和谨慎性降低到事故发生的合理因果链的支持。我们理解一个合理的因果链有助于证明我们得出原因与结果相关的结论。如果我们无法理解因果链，人们必须怀疑所声称的因果关系是否可能。

但我们了解因果链并不是我们证明推断因果概括的绝对必要条件。我的朋友得出的概括，"在发动机运转时转动钥匙导致巨响"，是由反复关联和刻意引入自变量证明的。这些东西似乎排除了对现象的任何替代解释。我的朋友对因果链一无所知。但尽管如此，如果得到关于连接转动钥匙（当发动机运转时）到巨响的合理因果链知识的支持，概括会得到更好的确立。

#### 来自更广泛因果概括的支持

第六种支持来自更广泛的因果概括和规律。例如，更广泛的概括"减少汽油发动机的空气和燃料供应会导致发动机减速"在给定一些合理假设的情况下，支持更狭窄的因果概括"松开油门踏板使汽车减速"。因果概括通常可以从更广泛和更基本的因果概括中推导出来。

#### 来自可信来源断言的支持

对因果概括的第七种支持可以来自可信来源的陈述（见第3章）。我们对许多因果概括的信念是以这种方式得到证明的。例如，我相信疟疾是由按蚊传播的生物引起的，这主要基于它来自可信来源。我没有对这个概括进行独立调查，也不熟悉任何支持它的证据，尽管毫无疑问有很多证据。

#### 摘要

所以至少有七种支持因果概括的基本类型：一般概括的单称因果实例的存在、实例的代表性、自变量和因变量之间关联实例的存在（在每个关联中效果没有发生在原因之前）、自变量被故意引入、对介入因果链的理解、因果概括可从更广泛的因果概括和规律中推导出来（给定合理假设），以及它们被可信来源断言。这些支持类型依赖于诉诸第8章引入并在本章扩展到非实验假设的假设检验四个基本准则。这种依赖甚至适用于通过表明来源可信来提供支持（还有什么其他方式可以解释可信来源的断言呢？）。可信性也独立于这种依赖而计数。

### 单称因果陈述

单称因果陈述的形式是"*X* 导致了 *Y*"或可以转化为该形式的某物，如"*X* 是 *Y* 的原因"。以下是一些例子：

> 刺伤导致了艾尔的死亡。
> 她实施了导致他死亡的行为。
> 狒狒的消失是由狮子的接近引起的。
> 错误的维护程序是坠机的原因。
> 她尖锐的回复引起了一阵笑声。

单称因果陈述是过去时态的，是关于一个序列的。它们不明确地超越证据；也就是说，它们不明确地断言关于未研究案例的某事。例如，我的第一个例子不明确地断言关于下一个被刺伤的人死亡可能性的任何内容。有理由认为这个单称因果陈述背后有一些因果概括或规律，但单称陈述本身并没有告诉我们这些是什么。

单称因果陈述受最佳解释推理准则的约束。阿琳刺伤艾尔导致他死亡的结论是解释他死亡事实的最佳解释。没有看似合理的替代解释经受了侦探和病理学家的审查。（支持这一结论的更完整论证在第2章作为检察官总结陈述的节选给出。）

#### 后此谬误

虽然原因必须发生在（或与）其结果同时）其效果之前，但一个常见的错误是认为仅仅因为一件事紧接着另一件事（也许是戏剧性地），第一件事就导致了第二件事。如第8章所指出的，以这种方式推理被称为*后此谬误*推理，这是谬误的。例如，在一场足球比赛中，马丁说他希望主队输。紧接着，客队触地得分。根据他朋友的说法，触地得分是由马丁希望输球引起的。如果结论仅基于触地得分紧跟着表达希望这一事实，那么这种推理就是后此谬误推理——而且是谬误的。

这种关系可能是巧合的。或者，两件事可能都是由其他东西引起的，也许是教练对主队四分卫的严厉话语，马丁无意中听到的。严厉的话语可能导致马丁希望球队输球，只是为了向教练证明，严厉的话语也可能使主队成员如此心烦意乱，以至于他们允许了触地得分。但无论是在巧合情况下还是在共同原因情况下，说马丁希望球队输球导致了触地得分都是不正确的。

#### 部分原因和条件

总是有大量的条件共同作用产生一个效果。通常我们选择其中一个条件并称之为*该*原因或*部分*原因。在原因、部分原因和条件之间做出区分取决于我们的兴趣和实际情况。在审判中，受害者死亡中唯一最重要的因素是故意刺伤，所以它被选中，尽管还有许多其他重要的条件，如艾尔没有穿厚皮饰钉夹克和刀的锋利。有时，因为我们认为不止一个条件是重要的，我们称它们为部分原因，称每一个为*一个*原因，而不是*该*原因。例如，如果艾尔曾被警告要警惕阿琳，每次和她在一起都要穿皮饰钉夹克和链甲背心，那么我们可能会说他没有这样做是他死亡的（部分）原因，而刺伤只是另一个（部分）原因。

你怎么知道该称某物为条件、部分原因还是原因？这个问题没有简单直接的答案，但作为异常发生（刺伤是异常的）、偏离应该做的事情（刺伤是）、故意的人类行为（刺伤也是）或给定其他条件的必要条件（刺伤也是），主张判断某物为原因而不是条件。

最后提到的考虑（在情况下是必要条件）被用于法律中的*若无*测试。应用于刺伤，它是这样的："**若无**刺伤，艾尔就不会死。所以刺伤是原因。"不幸的是，这个测试并不能确定地识别原因。关于艾尔的出生，我们可以真实地说，"若无他的出生，他就不会死。"然而，他的出生并不被认为是他的死亡的原因。此外，*若无*测试甚至并不总是给我们某物成为原因的必要条件，因为效果可能以某种方式被过度决定，以至于原因的不发生可能带来导致效果的东西。使用迈克尔·斯克里文建议的例子，河上游堤坝的失效导致了镇上的洪水。但如果堤坝被加固到没有失效的程度，水流的涌动会在那一点上经过堤坝，并在靠近城镇的地方冲毁另一道堤坝，城镇无论如何都会被淹没。所以堤坝的破裂甚至不是一个必要条件，尽管它导致了洪水。关于这个和类似的例子还可以说很多，但就目前的目的而言，这应该足够了。

如果一个条件比其他条件重要得多，给定情况和我们的目的，那么它很可能被选为原因。重要性，给定情况和目的，是我当地报纸头条的背景："塔架维护，而非设计，导致了DC-10坠机。"一旦我们意识到总是有多个条件参与产生一个效果，总是拒绝选出一个是具有误导性诱惑的。但刺伤是艾尔死亡的原因。鉴于情况，我们有理由选出那一个，并称其为*该*原因。

总之，偏离常态、偏离应该发生的事情、某物是故意的人类行为，以及某物是必要条件，这些都是在从条件集合中选出一个或多个原因时，以及决定某一个是*该*原因还是仅仅是部分原因时要考虑的准则。你必须运用你对情况的理解和你的良好判断力来应用这些准则。没有一个是某物成为原因的绝对必要条件（尽管*若无*条件——必要条件——通常成立）；相反，它们是在给定情况下应用判断力来发展或评价单称因果陈述时可以应用的准则。

将这些准则应用于阿琳那一刀导致艾尔死亡的案例：刺伤人是人们相互关系的正常方式的偏离。刺伤他是在陪审员眼中不应该做的事情。刺伤是故意的。如果他没有被刺伤，他在那时就不会死。因此，阿琳那一刀导致他死亡的主张至少满足第一、第三和第四个准则——而且对陪审团成员来说，第二个也满足。

#### 作为识别原因策略的替代方案消除

有一种在我们认为我们知道所有看似合理的或更好的所有可能原因的情况下识别原因的基本策略。它是调查并尽可能多地排除。如果只剩下一个，那么它可能就是原因。你会认出这个策略是上一章末尾讨论的主要故障排除和诊断策略。如果我们知道所有可能的原因，这个策略效果最好，即使如此，我们也应该谨慎，往往需要在因果结论中添加诸如*可能*之类的限定词。消防队长在追求这一策略时说，关于一场酒店火灾："我觉得不太可能是马达引起的。如果我们排除了那个，我们就排除了所有意外原因。那就只剩下有人纵火。"队长包含限定词的方式是使用*会*这个词，而不是明确地说原因是纵火。这种基本的消除策略在我们了解某一发生的所有看似合理或可能原因的情况下，为替代假设消除准则的直接应用。

当我家里的任何一盏灯不工作时，我使用这种策略的变体。自从我住在这里以来，每当任何一盏灯不工作时，总是因为灯泡烧坏了、开关坏了、断路器跳闸了或停电了。如果一盏灯停止工作，它的开关行为和平时一样（声音和动作），同一电路上的其他灯都亮着（排除了断路器跳闸和停电），那么最佳解释可能是灯泡烧坏了。最实际的做法是更换灯泡，而不检查每一个可能的替代假设（比如电线断裂），因为更换比进行替代假设的彻底检查要容易和便宜得多。如果更换灯泡不起作用，那么就需要更仔细地检查替代假设，因为最有可能的那个（灯泡有缺陷假设）有强有力的证据反对它。这种简化策略在通常只有一种原因的情况下，以及在补救措施出错不会花费太多的情况下是可以的。

关键是有时有实际理由不去检查每一个看似合理的可能假设。在这种情况下，有缺陷灯泡假设通常是对的（给定标准开关行为和其他灯亮着），合理做法是暂时当作它是真的。如果灯泡非常昂贵，而且拆下来就会损坏，或者更换起来昂贵或耗时（或者如果更换错了会危及生命，就像误诊可能导致的那样），那么在更换灯泡之前检查替代假设可能是明智的。但我的情况不是这样。我的灯泡很容易更换。

有时情况甚至不那么复杂。当只有一种东西可以导致发生的效果时，那么当这种效果发生时，我们就知道是什么导致了它。例如，如果只有极端高温才能使钢材以特定模式变成指定颜色，那么我们就有了确凿的证据证明这块钢材以该模式变成该颜色是由极端高温引起的。

#### 因果链

在得出单称因果结论时（甚至比因果概括更甚），理解原因带来效果的因果链是可取的，有时是必要的。病理学家得出那一刀导致艾尔死亡的结论，需要病理学家理解死亡是如何发生的（也就是说，理解从刺伤努力到刺入到失血到缺氧到死亡的因果链）。错误塔架维护导致DC-10坠机的结论，需要追溯类似这样的因果链：从维护程序到破裂的塔架，到发动机的丢失，到襟翼的丢失，到失速，到坠机。

另一方面，理解因果链并不总是必要的。正如我之前指出的，一个朋友不理解他的汽车在发动机已经运转时转动钥匙产生可怕噪音的机制。但他仍然正确地得出结论，他上次这样做时转动钥匙导致了那种噪音。他有三种证据支持这一结论：一种是有一系列场合，当发动机运转时转动钥匙立即伴随着类似的声音。这两种事件一直一起发生。第二，噪音不是按照任何时间表产生的，但总是在他转动钥匙时发动机运转时产生。他想不出有什么东西既能导致他转动钥匙又能导致噪音（所以可能不存在共同原因）。似乎没有其他解释可以解释这一系列否则难以置信的巧合，除了当发动机运转时转动钥匙总是导致噪音（在标准条件下）。他的单称因果结论是他这种因果概括的一个实例。

第三种证据涉及他上次这样做的情况，这是单称因果结论的内容。那次噪音正好在他转动钥匙时发生，据他所知，当时没有其他值得注意的事情发生。所以，给定当发动机运转时转动钥匙可以而且总是（在标准条件下）导致他听到的噪音，而且给定他当时看不到其他事情发生，那么他就有理由得出结论，那次转动钥匙导致了他听到的噪音。即使他不理解连接转动钥匙和噪音的机制，他也有权得出这个结论。

注意这不是*后此*谬误的情况。证据比两个事件仅仅一次相继发生要多得多。

还要注意，如果我们不理解因果链，单称因果结论需要更强的证据。如果我们确实理解因果链，论证会更容易。如果不理解因果链，汽车噪音证据必须比如果理解了因果链所需的要强得多。

我提醒你人类刻意*引入*被怀疑是原因的因素这一策略的重要性。如果我们随意引入那个因素（而且它的发生立即伴随着效果），那么往往很难想出对我们做的两件事的解释，除了我们做的事情导致了另一件事。

#### 因果链中的断裂

理解因果链以另一种方式与因果结论相关。如果我们知道一个发生带来效果的唯一因果链（或链条），那么如果该链（或那些链）中存在断裂，该发生就不是原因。在酒店火灾中，确定圣诞树灯没有插上插头，所以产生圣诞树灯短路热量所需的电力不可用。所以圣诞树灯短路被排除了。因果链中存在断裂。

排除某物作为原因的另一种方法是表明它没有发生或不存在。表明房间着火时没有点燃的蜡烛存在，就会排除点燃的蜡烛作为原因。

#### 可信来源

确立单称因果陈述的一个常用方法是表明它被可信来源相信。病理学家的证词，即那一刀导致了艾尔的死亡，被接受为确立这一事实。

### 摘要与评论

虽然因果关系是一个困难且棘手的话题，但我们可以运用松散的标准来做出和评判因果结论。以下总结的标准本身并不能产生正确答案。应用这些标准需要良好的判断力、作为理由（*R*）和反理由的事实背景知识，以及对情况（*S*）的敏感性。基本标准是最佳解释推理的标准（*I*），特别关注"是否有另一个看似合理的解释？"这一问题。

这些标准确实为你的判断力的运用提供指导。然而，有些人如今建议的那样，无法判断一件事是否导致或将导致另一件事，这不是真的。通常我们无法做到这一点，但通常我们能够做到。我们一直在这样做，如果我们遵循指南，往往是正确的。但我们无法确定一件事是否导致了另一件事的挑战，在困难的情况下往往是适当的，比如确定拿破仑是否死于砷中毒，以及确定总统在给定时间知道什么和意图是什么。

因果概括*明确地*适用于不是其证据一部分的案例。但必须理解它们隐含地包含模糊的限定*在标准条件下*，而且它们往往没有指明肯定案例的大致比例。然而，这种指明可以用诸如*总是、几乎总是、通常、经常、有时*和*可以导致*之类的术语做出。在陈述因果概括时，我们很可能包括这样一个限定术语，在决定是否相信一个时，我们首先需要确定其普遍性程度。

对因果概括的支持基于其满足假设四个一般准则的能力。更具体的支持至少有七种类型，可以通过诉诸这些准则来证明，但在这里列出它们是有用的。不要背诵这些。只要确保你能想到每个的例子。

1. 一般概括的单称因果实例的存在。
2. 实例的代表性。
3. 假设原因案例与假设效果案例关联的一组实例的存在，即使在任何一个案例中没有确立*X*导致了*Y*。在每个案例中，效果不能发生在原因之前。
4. 研究者故意引入了自变量（即假设的原因）。
5. 因果链的合理性。
6. 因果概括可从一组或多组更广泛的因果概括或规律中推导出来。
7. 因果概括被可信来源断言。

单称因果陈述可以，但不必，在实验的基础上做出。关键问题是它们是否解释事实，以及是否有另一个对事实的看似合理的解释。换句话说，假设的四个基本准则也适用于此。对单称因果陈述的更具体支持至少有七种类型，可以通过诉诸四个一般准则来证明，但讨论和单独列出它们是有用的。同样，不要背诵这些。只要确保你能提供每个的例子。

1. 消除所有看似合理原因集合中除一个以外的所有，或者更好的是，消除所有可能原因集合中除一个以外的所有。
2. 清楚理解假设原因可能带来效果的因果链。
3. 确保假设原因发生了，而且它没有发生在效果之后。
4. 表明假设原因是唯一可以导致发生的效果的那种东西的一个实例。
5. 表明这四个粗略准则中的一个或多个得到满足：假设原因是异常发生，是错误或不当的，是故意的人类行为，或在情况下是必要的（*若无*条件）。
6. 表明单称因果陈述是一个确立的因果概括的实例。
7. 表明单称因果陈述是由该陈述的可信来源做出的。

*后此推理*是我们仅基于效果发生在假设原因之后而得出单称因果结论的推理。虽然这种推理是谬误的，但某件值得注意的事情发生在效果之前的事实，可以是支持值得注意的事情是原因的单称因果陈述的一组证据的一部分。

单称因果陈述并不隐含普遍因果概括。此外，证明一个普遍结论为假并不能表明可以从它推导出的单称结论为假。例如，表明使用批判性思维书籍在一个案例中导致了提高，并不能表明它总是或将会如此。而且表明使用这本书并不总是如此，也不能证明它在某些特定情况下不会如此。

这种对因果关系的讨论基本上是发展基于最佳解释推理原则的有用因果推理指南。这些原则对于那些想知道如何开始进行因果调查的人是有用的。对于那些理解ably 对得出因果结论持谨慎态度的人，这些原则和讨论也应该是有用的提醒，即我们能够而且确实在日常生活中合理地得出因果结论。在因果推理中很容易出错，但我们也能把它做对。

关于*原因*这个概念，还有更多要说的和学的。这是一个你可以在此基础上建立完善的开始。

## 立场论文和其他论证性写作中的最佳解释与因果推理

在许多论证性写作中，我们使用最佳解释和因果推理。通常我们的最终结论本身就是最佳解释结论，就像历史学家支持拿破仑死于砷中毒结论的书面论证中那样。但通常最终结论本身不是最佳解释或因果结论。相反，一些最佳解释或因果结论是作为中间结论提出的，以支持通过某种其他类型推理达到的最终结论。

你在本书两个不同地方看到过的一个这样的写作例子是布拉德利关于伊阿古不是一个通俗剧式反派的论证，至少在我对他的论证的理解中。在本章前面，你看到了一些支持爱米利娅不怀疑伊阿古是恶棍结论的最佳解释推理。这个结论是一个更大论证中的中间结论，因为它在第7章提出的演绎论证中充当了理由，导致最终结论伊阿古不是一个通俗剧式反派。这个最终结论不是最佳解释结论，而是由一个最佳解释结论支持的。此外，伊阿古不是一个通俗剧式反派的最终结论本身可以成为用于支持价值结论的中间结论，"扮演伊阿古的人应该在剧中几乎所有人面前表现得和举止都很正常。"（顺便说一下，这个结论在所有情况下都不仅仅是一个微不足道的补充。一些表演者和导演可能想做得不同，把伊阿古演得像一个通俗剧式反派来挑战现在的传统。或者，一些崭露头角的演员可能不知道通俗剧式反派是什么样的，也可能没有意识到通俗剧式反派不会表现得和举止都很正常。）

导致价值结论（如"我们不应该使用DDT来杀死昆虫"）的论证通常利用中间最佳解释结论（如"DDT削弱鸟蛋壳"）。DDT削弱鸟蛋壳的结论是一个因果结论（*削弱*是一个因果词，如果像我一样广义地理解*因果*），或至少是一个最佳解释结论，作为价值结论的理由。大多数价值论证都有一个或多个这种类型的理由来支持它们的结论。这些理由应该是可以辩护的。辩护可以来自可信来源的证词（第3章讨论）、最佳解释论证（本章和前几章讨论），或两者结合。

### 一般写作策略

有时你论证中的理由是你觉得可以期望你的受众接受的基本假设。或者，它们可能是你做出的观察、他人的证词，或你在以前的论证中得出的结论。在任何情况下，都要意识到它们以及你对它们的任何辩护（如果有的话）。如果你对其中一个没有辩护，要诚实并注意你的论证确实依赖于它。

在你的论证过程中，要牢记情况和受众的观点，即你为之写作的人或人们。你选择详细说明和辩护的理由，以及你选择无需辩护的假设，部分取决于你的受众。在呈现你的论证时，你不仅试图提出一个好的案例，而且试图解决他们的关切。如果你的受众已经接受某事是显而易见的，通常浪费他们和你的时间去证明给他们看（尽管你可能这样做是为了向他们指出他们的一些信念确实需要辩护）。不断问自己，"这对我的受众有意义且有帮助吗？"

对于论证的最佳解释部分，提出你想到的或他人提出的替代解释。诚实地处理这些，指出它们的弱点和优点。指出你拒绝它们的理由（如果你拒绝）。指出任何仍然具有合理性的。如果有对你证据的挑战，指出并处理它们。如果有任何看似合理的，要说清楚。

当你写一篇结合各种类型论证或由几个部分组成的立场论文时，你必须想出一个合理且可理解的顺序来安排这些部分。最简单和最直接的形式是，在写一页或更多页的论证时，你先介绍情况和你的最终结论，可能还要总结整个论证。你以重述你的最终结论结束立场论文，提及你论证的各个部分。我向时间紧迫的读者推荐这种简单直接的形式，通常当写作由非专业作家完成时。

对于中间的论证填充，或对于简短的论证，有两种主要的直接策略，有许多变体，我将首先讨论。我称它们为*自下而上*和*自上而下*策略。当你阅读这些策略时，想一些你写过或计划写的特定立场论文或文章，并考虑它在多大程度上体现了这些策略。

#### 自下而上策略

第一种策略是通过从基本理由开始来构建你的论证，这种基本理由出现在你第2章图表的底部。（我建议你现在回过头去看第2章末尾建议的图表答案，如果你没有理解这一点的话。）在这些基本理由上，你一次构建一个或多个中间结论。然后你将中间结论放在一起，发展建立在它们之上的论证，对你的论证主张不超过它应得的强度。第2章中的项目 2:51（男性敏感性）、2:52（女性自信心）和 2:67（SAT分数）体现了自下而上策略（先理由，后结论）。

#### 自上而下策略

与自下而上策略相反的是自上而下策略，在这种策略中，你先给出你的最终结论，然后是你的理由，以某种方式表明你打算为其理由辩护（或者如果必然是一篇短文，你的理由是可以辩护的）。然后你继续为理由辩护，每个理由都有论证支持。布拉德利关于爱米利娅从未怀疑伊阿古是恶棍的论证（本章例 9:3）体现了简化的自上而下策略，第2章中的项目 2:66（使用动物的研究）也是如此（先结论，后理由）。

#### 组合策略

你可能猜到了，这两种基本策略有各种可能的组合。第2章中的例 2:61（学校的功能）和 2:68（利益冲突）体现了两种策略的组合（理由、结论、更多理由）。第2章中检察官的演讲体现了适合较长论证的不同组合策略（最终结论、一系列导致中间结论的论证、重复最终结论）。

#### 间接策略

有些情况需要各种间接策略之一。例如，有时你可能不明确陈述结论，就像第2章中的项目 2:62（家庭作业）和 2:65（路灯）那样。不陈述你的结论有时会使写作更有趣。有时写作会变得更有效，因为读者必须推断或构建结论，然后觉得结论是他或她自己的结论。有时这种写作更谨慎，因为在情况下，读者会觉得你在直接激进地陈述结论时过于唐突。这后一种情况在人们期望不要出风头的文化中很常见。

不明确陈述结论的一个主要危险是，你的受众不会理解你的结论——或者，更严重的是，会误解你，并让你为你不持有的观点负责。偶尔的危险是，时间紧迫的受众就是不想花时间理解它。也许受众应该愿意花时间，但许多人不愿意。此外，通常他们不够专业，无法得出适当的结论，需要帮助。

不陈述结论可能是一种欺骗性技巧。这是一种逃避为你想得出但确实不能很好辩护的结论负责的方式。那么你总是可以说，"我从没那样说过！"另一方面，如果结论是正当的，但你会因为断言它而从狭隘的读者那里惹上麻烦，避免陈述它可能是有用的——只是为了避免麻烦，这在某些制度下可能很严重：可能是监禁、处决或酷刑。

另一种间接策略是绕开问题，有时引入不相关的材料（也就是说，有趣且与主题相关的材料，但不支持结论或其否定，也不帮助表明结论的重要性）。这种材料可以增加兴趣，而且像避免陈述结论一样，可以帮助避免指责作者通过擅自建议什么是相关的什么是不相关的来出风头。

绕开问题可以很好地服务于这些目的。但危险在于一些读者会错过论证，并对所需的额外时间失去耐心。对于其他读者来说，这些不是问题。

### 摘要

你必须在分析情况后选择你的策略，包括你的受众。准备好根据受众的接受情况改变你的策略。在任何情况下，都要意识到你自己和受众可能的假设。确保你自己的假设被澄清，他们的假设被解决。如上所述，注意并处理替代假设和对你证据的可能挑战。确保对你的论证主张不超过（或低于）它应得的强度。

最直截了当的立场论文写作策略是呈现你的结论，讨论其重要性和你的论证计划，提供你的理由，考虑反论证，并总结。其他策略有优点和缺点，最常见的缺点是无法传达你的观点。
